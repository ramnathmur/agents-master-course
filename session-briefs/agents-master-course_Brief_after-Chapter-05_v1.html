Chapter 00 · The Toolsmith · Trait: None. Plumbing only.

Setup: Claude Max + SDK + IntelliJ

I worked through Setup: Claude Max + SDK + IntelliJ. The chapter focused on none. plumbing only.. What I'm carrying forward: The SDK shells to the CLI. Configuration precedence is silent. Verify your env, not your assumption.

  • **The SDK is a CLI wrapper.** `claude — agent-sdk` is not a fresh HTTP client. It shells to the `claude` CLI binary that `pip install` brought along. That CLI owns the OAuth session. Your Python code never sees an API key on the Max path.
  • **Auth precedence is silent, not loud.** If `ANTHROPIC_API_KEY` is set, the SDK uses it without a warning. There is no "you have two auth sources, which one?" prompt. The architect's discipline — explicit `unset` or empty value in every run config you create.
Carry-forward: The SDK shells to the CLI. Configuration precedence is silent. Verify your env, not your assumption.
